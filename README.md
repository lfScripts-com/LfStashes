EN: Thank you for using the LfStashes script!

Installation
1. Place the script in your resources folder.
2. Add `ensure LfStashes` to your server.cfg file.
3. Configure the script via `config.lua`.

The script handles stash creation and management through ox_inventory, including a NUI interface, ESX permissions and JSON storage. If you have any questions or need help, we are here to assist you.
